### TCP vs UDP
TCP:
- Reliable
- Data arrives in order
- Used for websites and email
- waits for each packet to arrive
  
UDP:
- Faster
- No guarantee of delivery
- used for streaming and gaming
- He's a trooper - he keeps going with or without all the pieces
